# Kindling documentation (Docusaurus)

This site documents the **Kindling** Python package in the repository root.

## Commands

| Command | Description |
|---------|-------------|
| `npm install` | Install dependencies |
| `npm start` | Dev server (usually http://localhost:3000) |
| `npm run build` | Production static build to `build/` |
| `npm run serve` | Preview the production build locally |

## Content

Markdown and MDX files live in `docs/`. Sidebar order is defined in `sidebars.ts`.

Update `docusaurus.config.ts` (`url`, `baseUrl`, `organizationName`, `projectName`) when you publish the site.
